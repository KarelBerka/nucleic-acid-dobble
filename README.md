# 🔬 NucleicAcid-Dobble (NA-Dobble)

Výuková karetní hra Dobble zaměřená na nukleové kyseliny, báze, cukry, nukleosidy, energetické nukleotidy a tRNA speciality.

🌐 **Živá aplikace (GitHub Pages):** [https://karelberka.github.io/nucleic-acid-dobble](https://karelberka.github.io/nucleic-acid-dobble)

---

## 🎮 Herní verze

1. **NADobbleMini ($q = 2$)**
   - 7 symbolů, 7 karet, 3 symboly na kartě.
   - Obsahuje 5 základních bází (A, T, U, C, G) + 2 cukry (Ribóza, 2-Deoxy-D-ribóza). Ideální jako rychlá rozcvička.
2. **Plná verze ($q = 4$)**
   - 21 symbolů, 21 karet, 5 symbolů na kartě.
   - Báze, nukleosidy, nukleotidy (AMP, CMP, GMP, UMP, dTMP), energetické nukleotidy (ATP, GTP, cAMP) a tRNA speciality (pseudouridin, inosin, dihydrouridin).
